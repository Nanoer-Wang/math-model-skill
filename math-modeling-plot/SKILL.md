# 数学建模绘图 Skill

## 目标
生成清晰、美观、适合论文展示的图表。图表应解释数据、支撑模型、展示结果，而不是单纯装饰。

## 触发场景
- 用户要求画趋势图、热力图、空间地图、散点图、箱线图、雷达图、网络图、流程图。
- 用户要求美化建模论文图表。
- 用户需要模型框架图、算法流程图、结果对比图、敏感性分析图。
- 用户需要保存高分辨率 PNG/SVG/PDF。

## 不应触发
- 只需要训练模型时，转交代码 Skill。
- 只需要排版 Word/LaTeX 时，转交排版 Skill。

## 输入
尽量收集：
- 数据文件或结果文件路径。
- 图的目的：说明背景、验证模型、比较方案、展示预测、展示优化结果。
- 目标格式：论文、答辩 PPT、海报、附录。
- 字体、颜色、大小、分辨率、是否中文显示。

## 工作流
1. 明确图表目的：每张图对应一个论文结论。
2. 选择图型：趋势用折线，分布用箱线/小提琴，相关性用热力图，预测效果用真实-预测散点和残差图，优化路径用收敛曲线。
3. 准备数据：排序、聚合、标准化、长宽表转换、异常值标注。
4. 设计图形：标题简洁，轴标签包含单位，图例不遮挡数据，颜色控制在 3–6 种。
5. 保存文件：同时保存图片和绘图数据，便于复现。
6. 输出说明：说明图中关键结论，可直接写进论文。

## 常用图表建议
- 数据概况：缺失值热力图、变量分布图、相关系数热力图。
- 时间序列：原始序列、平滑趋势、预测区间、残差序列。
- 机器学习：特征重要性、真实值 vs 预测值、残差分布、误差箱线图。
- 优化问题：收敛曲线、方案对比柱状图、Pareto 前沿。
- 评价问题：指标权重图、TOPSIS 得分图、雷达图。
- 空间问题：地图、空间热力图、OD 流向图。
- 论文结构：技术路线图、模型流程图、算法框图。

## Python 绘图规范
- 优先使用 matplotlib、pandas、networkx、plotly、geopandas。
- 图像尺寸和 DPI 显式设置。
- 中文字体要做兼容处理；若环境缺字体，使用英文图题或提示安装字体。
- 每张图单独保存。
- 同步保存 `.png` 和 `.svg`，关键论文图建议再保存 `.pdf`。
- 保存绘图源数据到 `figures/data/` 或 `outputs/plot_data/`。

## 输出格式
默认输出：
- 绘图代码。
- 图片文件路径。
- 绘图数据文件路径。
- 图注草稿。
- 图中结论解释。

## 质量标准
- 图表必须能支持论文叙述。
- 坐标轴、单位、图例、注释必须完整。
- 不要使用过多颜色、3D 饼图、无意义装饰。
- 不要截断坐标轴造成误导，除非明确标注。
- 图注应说明数据来源、变量含义和主要发现。

## 禁止事项
- 不要只生成默认样式且无解释的图。
- 不要在图中堆满文字。
- 不要把不同尺度的数据强行画在同一坐标轴而不说明。
- 不要用无法复现的手工绘图替代代码绘图。
