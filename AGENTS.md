# 数学建模项目 Codex 指令

当任务涉及数学建模竞赛、数学建模课题、CUMCM、MCM/ICM、研究生数学建模、课程论文、模型求解、数据分析、可视化或论文排版时，优先调用对应的 math-modeling-* skill。

## Skill 调用建议

- 文献/背景/研究现状/参考文献：使用 `math-modeling-literature`。
- 数据获取/爬取/清洗/模拟/字段字典：使用 `math-modeling-data`。
- 模型假设/变量/公式/算法路线/创新点：使用 `math-modeling-modeling`。
- Python/MATLAB 实现/调试/运行/保存结果：使用 `math-modeling-code`。
- 图表/流程图/结果图/敏感性分析图：使用 `math-modeling-plot`。
- 论文正文/摘要/结果分析/优缺点评价：使用 `math-modeling-writing`。
- Word/LaTeX/PDF/参考文献/最终提交检查：使用 `math-modeling-typesetting`。

## 总体约定

- 中文输出优先。
- 真实数据优先，模拟数据必须明确标注。
- 所有外部数据、文献、政策和网页来源必须记录。
- 代码必须可复现，保存随机种子、参数和输出文件。
- 论文结果不能编造，文字必须与数据和代码结果一致。
